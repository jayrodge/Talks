# Intro to Machine Learning using PHP

## Description
This talk was delivered at [Chicago PHP](https://twitter.com/ChiPHPUG) [Meetup](https://www.meetup.com/Chicago-PHP-User-Group/events/262976448/)

In this talk, a Introduction to Machine Learning was given for beginner audience and discussed a Machine Learning Library for PHP called [Rubix ML](https://github.com/RubixML) developed by [Andrew DalPino](https://github.com/andrewdalpino)

A Classification problem for Iris Dataset using K Nearest Neighbour which can be accessed at [Repo](https://github.com/RubixML/Iris)
